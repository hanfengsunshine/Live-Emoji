#### App

Pre-authoring for 2D live animation

#### Implemented

* Import OBJ
* Node Editor UI
* Image Triangulation
* Face tracking
* Asset

#### How to use

* `npm install`
* `git clone https://github.com/priologic/easyrtc.git`
* `cd easyrtc`
* `npm install`
* `cd ..`
* `node server.js`
* `https://localhost:8080/`, `https://localhost:8080/robot.html`

#### Dependencies

* three.js r89
